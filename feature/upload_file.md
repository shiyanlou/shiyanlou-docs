# 上传文件（会员）

## 介绍

上传文件功能可以将本地的文件上传到实验环境中，主要的应用场景为需要向环境中传输一些本地的代码及数据文件。上传文件是实验楼会员功能。

![ssh](https://doc.shiyanlou.com/shiyanlou-docs/images/upload.png)


## 使用

开始实验后，在学习界面中点击右边工具栏上的 `上传文件` 按钮后会弹出上传文件提示框。

选择上传的文件，然后点击窗口右下角的 `上传` 按钮。文件就会被上传到实验环境中的 `/home/shiyanlou/Code` 目录下。

需要注意上传的文件大小不能超过 5M。
